Immune Cell Classification

File: [sample_name]_(assay)_cell_type_experimental.csv

These files contain the immune cell classification results. The classification is based on the expression of specific genes and is used to identify different immune cell types. We collected a large dataset of PMBCs that had targeted immune response panel (397 genes) and AbSeq expression. Performed dimensionality reduction. We used following cell types for classification:

  • Classical monocyte
  • Nonclassical monocyte
  • Dendritic cell
  • B cell
  • Naive CD4 T cell
  • Memory CD4 T cell
  • Naive CD8 T cell
  • Memory CD8 T cell
  • Natural killer cell
  • Gamma-delta T cell

If customer using other than PBMCs, it is recommended to apply your own cell type classification in the downstream analysis.

When the Immune Cell Classification is done:

  • Using WTA/AbSeq cell calling data: [sample_name]_cell_type_experimental.csv
  • using ATAC cell calling data: [sample_name]_ATAC_cell_type_experimental.csv
  • using WTA/AbSeq and ATAC cell calling data: [sample_name]_Joint_cell_type_experimental.csv
