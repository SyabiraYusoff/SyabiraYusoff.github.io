Sample Tag Calls

File:
[sample_name]_Sample_Tag_Calls.csv
  • The Sample Tag calls file contains the determined sample call for every putative cell.
  • The Sample Tag calls file can be used to annotate the main data tables, which contain results from all samples.
  • Sample names that you provided are included in a separate column. For example:
    Column Description
    Cell_Index Unique cell identifier
    Sample_Tag The sample tag identifed for the cell
    Sample_Name User-provided corresponding sample name
