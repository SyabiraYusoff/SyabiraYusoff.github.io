VDJ Per Cell

File:
[sample_name]_VDJ_perCell.csv

  • This file is only output when the experiment included an appropriate TCR/BCR assay, and the VDJ_Version option is selected.
  • This file contains the VDJ data for putative cells in the sample.
  • The data is organized by cell index, the same as gene expression data table (RSEC/DBEC_MolsPerCell_MEX.zip).
  • Only dominant contigs is included, with all correction applied.
The data columns below:
  • Read and molecule counts
  • VDJ gene segments
  • CDR3 sequence
  • Pairing
  • Cell type
