Bioproduct Statistics

Bioproduct Statistics:

The molecular identifier adjustment algorithms RSEC and DBEC are applied to each bioproduct. The molecular identifier metrics file lists the metrics from RSEC and DBEC on a per-bioproduct basis. For more information on RSEC and DBEC molecular identifier adjustment algorithms, see Molecules and error correction.

File: [sample_name]_Bioproduct_Stats.csv

Bioproduct Statistics Information:
Metrics Definition
Pipeline Analysis Information Bioproduct status across all reads and molecules:
  • Pipeline Version
  • Analysis Date
  • Type of Bioproduct libarries detected
  • Reference Input
  • Parameters settings
Bioproduct Bioproduct names listed in alphabetical order
Raw_Reads Number of reads before molecular identifier adjustment algorithms
Raw_Molecules Number of UMIs before molecular identifier adjustment algorithms
Raw_Seq_Depth Number of raw reads รท the number of raw molecules
RSEC_Adjusted_Molecules Number of molecules detected after RSEC molecular identifier adjustment algorithm
RSEC_Adjusted_Seq_Depth Number of raw reads รท the number of RSEC-adjusted molecules
RSEC_Adjusted_Molecules_nonsingleton Number of RSEC-adjusted molecules represented by more than one read
DBEC_Minimum_Depth Threshold of RSEC depth for a molecule to be considered a putative molecule by DBEC
DBEC_Adjusted_Reads Number of reads retained after DBEC molecular identifier adjustment algorithm
DBEC_Adjusted_Molecules Number of molecules retained after RSEC and DBEC
DBEC_Adjusted_Seq_Depth Number of DBEC-adjusted reads รท the number of molecules detected after RSEC and DBEC
DBEC_Depth_Status Bioproduct status across all reads and molecules:
  • Not detected: Bioproduct was not detected, because it has zero reads
  • Low depth: Minimum sequencing depth not achieved. DBEC not applied.
  • Pass: Minimum sequencing depth has been achieved
