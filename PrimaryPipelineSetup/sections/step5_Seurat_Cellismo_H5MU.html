Seurat and Other Single-Cell Analysis Files

File Types:
  • Seurat File: [sample_name]_seurat.rds
  • Cellismo / Scanpy / Muon File: [sample_name].cellismo

Popular Toolkits:

  • Seurat 4: satijalab.org/seurat
  • Cellismo / Scanpy / Muon: scanpy.readthedocs.io / muon.scverse.org

Experiment Types:

  • RNA and AbSeq Experiments:
    • Contains RSEC molecules-per-cell data table for putative cells
    • Includes cell and bioproduct metadata
  • ATAC-Seq Experiments:
    • Contains cell-by-peak data table for putative cells
    • Includes cell and bioproduct metadata

Metadata includes (if applicable):

  • Sample tag calls
  • Putative cell origin
  • TCR/BCR chain types and CDR3
  • Immune cell type (Experimental)
  • Protein aggregate
  • tSNE/UMAP coordinates

Multimodal Data:

  • Seurat: Built into separate "assay" objects ("RNA", "ADT", and "peaks")
  • MuData: ("rna", "prot", and "atac") respective data structures
