AbSeq Reference

AbSeq Reference is a feature that allows you to upload your own reference files to the BD Rhapsody™ Pipeline. This feature is useful when you have a custom AbSeq as well as pre-designed panels.

Instructions

  1. Navigate to BD AbSeq Panel Reference Generator
  2. If BD® OMICS-One Protein Panel(s) or BD® AbSeq Immune Discovery Panel is used in the experiment, click “<” on the left of the Panel (highlighted in the red box) to add to the “My Panel” column.
  3. If other BD® AbSeq Antibody-Oligo(s) are used in the experiment, select the BD® AbSeq product(s) and click “>>” to add to the “My Panel” column.
  4. After all antibody-oligo products are added to the “My Panel” column, fill in the “output file name” field with your preferred file name and click “Create New AbSeq Panel” to save the FASTA file to your computer.
