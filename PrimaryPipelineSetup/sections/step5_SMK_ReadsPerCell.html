Sample Tag Reads Per Cell

File:
[sample_name]_Sample_Tag_ReadsPerCell.csv
  • The Sample Tag reads per cell file contains the detected reads for every putative cell.
  • The Sample Tag calls file can be used to annotate the main data tables, which contain results from all samples.
  • All sample tags are included in a separate column. For example:
    Column Description
    Cell_Index Unique cell identifier
    Sample_Tag_Read_Count (column 02 - 13) The sample tag counts for the cell
