Pipeline Output Files Explained

General Output Files

These files are generated by the pipeline and contain important information about the analysis:

  • Metrics Summary
  • Bioproduct Statistics
  • Dimensionality Reduction Coordinates
  • Immune Cell Classification
  • Data Tables
  • BAM and BAM Index
  • Pipeline Report HTML
  • Single-cell analysis tool inputs
  • For a multiplexed samples run, theses tables contain counts for putative cells from all samples combined.
  • DBEC data table is only output if the experiment includes targeted mRNA or AbSeq bioproducts

When SMK Setting is Activated

When the SMK setting is activated, the pipeline will generate additional output files that are specific to the SMK analysis.

  • Sample Tags Metrics
  • Sample Tags Calls
  • Sample Tags Reads Per Cell
  • Sample Tags per Sample Folder

When VDJ Setting is Activated

  • VDJ Metrics Summary
  • VDJ Per Cell Metrics
  • VDJ Unfiltered Contigs
  • VDJ Dominant Contigs

When ATAC Setting is Activated

  • ATAC Fragments
  • ATAC Transposase Sites
  • ATAC Peaks
  • ATAC Data Tables
  • ATAC Metrics
