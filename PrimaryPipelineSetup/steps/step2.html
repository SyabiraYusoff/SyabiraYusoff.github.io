Upload Files to Velsera Platfrom

For Local Installation Users:

You may skip this step. You already have the files on your machine that you will use to run your local installation pipeline.

What is this Chapter for?

This chapter guides you on how to upload your files to the Velsera platform. There are multiple ways you can upload your files:

  • Upload directly on Cloud Platform:

    This approach is suitable for users who have the files on their local machine and want to upload them to the Velsera platform.However, it is recommended to upload your files through Command Line Interface (CLI) for files more than 1GB size.

  • Copy files from other projects:

    Some experienced users might want to use the same files or pipeline from other projects. This chapter includes a tutorial on how to copy files from other projects you have created. Please navigate to the Cloud Based Upload menu.

  • Upload through FTP or HTTPS:

    This approach is suitable when you have files from AWS or any other resource that is accessible through an FTP or HTTPS link. For example, reference files that are available publicly on a cloud platform.

  • Command Line Interface:

    This approach is useful when you have large files to upload and don't want to stay in front of your web browser. Please navigate to the CLI for MacOS or Windows.
