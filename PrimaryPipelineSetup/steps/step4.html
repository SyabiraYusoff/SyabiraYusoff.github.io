Run your BD Rhapsody™ Pipeline

Now that you have set up your pipeline, uploaded your sequencing data, and selected your reference files, you are ready to run the BD Rhapsody™ Pipeline. First, you need to select the menu on the right, whether you are running your pipeline on the Cloud Platform (Velsera) or Locally.

What is this Chapter for?

This chapter will show you how to set up your pipeline run:

  • Which sections to input your reads?
  • Where to input your specific reference files?
  • How to specify if you are using VDJ and/or SMK assays?

What does the "Other Pipeline Settings" menu contain?

This section contains other pipeline settings that you might want to check out in case you are interested:

  • To modify our cell calling algorithm.
  • Whether you want to exclude any intronic regions of your data.
  • For ATAC users, specific settings are available.
  • If you want to create a memorable Run Name setting for your pipeline run (good practice).
  • If you want to activate BAM settings and make it available as an output of the pipeline run.
