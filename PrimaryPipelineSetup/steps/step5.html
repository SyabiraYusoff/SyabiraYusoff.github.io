Pipeline Output Files

What is this Chapter for?

This chapter will show you the output files generated by the BD Rhapsody™ Pipeline. The pipeline generates a variety of files that are essential for QC as well as for downstream analysis.

What are the General Output Files?

The general output files are generated regardless of the assay type. These files are essential for QC and downstream analysis:

  • Bioproduct Statistics: This file contains metrics from RSEC and DBEC Unique Molecular Identifier adjustment algorithm on a per-biorpoduct basis.
  • Metrics Summary: This file contains report for sequencing metrics, molecules as well as cell metrics.
  • Dimentionality Reduction Coordinates: This file contains UMAP/tSNE for each cell label.
  • Immune Cell Classification: [Experimental] This file contains the cell type prediction per cell index. This is applicable for customers with PBMCs.
  • Pipeline Report HTML: This file contains summary report of results from the sequencing analysis pipeline.
  • BAM and BAM Index:[Available when activated on pipeline settings only] This file contain the alignment file of R2 with associated R1 annotations for Bioproduct.
  • Data Tables: This sparse matrix file contains Molecules per bioproduct per cell.
  • Seurat.rds: Input file for Seurat package in R.
  • Cellismo/H5MU: Input file for BD Cellismo™ secondary analysis tools. This file can also be used directly for Scanpy tool in Python.

What are the Assay Specific Output Files?

These files are tailored to the selected assays. We will look further into the content in further sections within this chapter.

  • VDJ: This file contains the VDJ annotation for each cell.
  • SMK: This file contains the SMK metadata for each cell.
  • ATAC: This file contains the ATAC annotation for each cell.
